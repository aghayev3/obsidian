# The OSI Model

![[Pasted image 20251119130137.png]]

---
1) Physical - data cables, cat6
2) Data Link - switching, MAC addresses
3) Network - routing, IP addresses
4) Transport - TCP/UDP
5) Session - session management
6) Presentation - WMV, JPEG, MOV (media)
7) Application - HTTP, SMTP, etc.
- Receive - Physical -> Application
- Transmit - Application -> Physical
- When troubleshooting start with the Physical Layer
